About This Site

Initially, this website was created as an exercise in testing out some new skills I learned. I may have learned the basics of CSS layout, but my methodologies and CSS structure is pretty bad. My next steps will be to first refactor my existing CSS to be more logical and readable. Once I've done that and learned some best practices, then I will move onto converting all my styles into Sass. I think that starting with vanillas CSS will allow me to get a better grip on the basics before I do more advanced things.

After getting a Sass library for the site, I hope to expand it a bit. I plan to create some more pages to showcase some of my work and act more like a portfolio. This will probably start as some small JavaScript projects so that I can learn about adding that final piece of the puzzle to this pure HTML and CSS site.
